__README__
==========

Данное приложение предназначено для обработки чисел из файлов формата xlsx.
Использованный стек: Java SE 21, Spring Boot 3.5.7, Gradle, Lombok, REST-архитектура, Swagger, Apache POI.

Установка и запуск:
-----------------------------------

**Вариант 1**

Склонируйте репозиторий.

На устройстве должна быть установлена Java. В директории ```statsfinder``` склонированного репозитория откройте GitBash 
и выполните команду:

```./gradlew clean build```

В директории ```statsfinder\build\libs``` появится jar файл. Откройте GitBash в этой директории 
и выполните команду для запуска приложения:

```java -jar [имя jar файла]```

Пример:

```java -jar statsfinder.jar```

**Вариант 2**

Склонируйте репозиторий. 

На устройстве должен быть установлен Docker. 

Откройте GitBash в директории ```statsfinder``` и выполните команду:

```docker-compose up -d```

Работа с приложением:
---------------------------------------

После запуска приложение будет доступно по адресу: http://localhost:8585

Приложение поддерживает Swagger. Получить доступ к Swagger UI можно по адресу:  http://localhost:8585/swagger-ui.html.

Приложение читает список чисел из файла формата xlsx  и выдает минимальное число, порядковый номер 
которого запрашивается в запросе. Дубликаты чисел учитываются. В файле разрешены только целые числа.
Файл должен содержать список чисел в первой колонке первого листа начиная с ячейки А1.

**Метод getMinNumberFromXlsx** - принимает запросы POST на:

```http://localhost:8585/api/file-scanner/find-min-number/xlsx```

В теле запроса нужно указать путь к файлу xlsx (или загрузить его если запрос делается через Swagger) 
и желаемый номер (целое число).

При корректном запросе метод возвращает JSON с кодом ```200``` и числом-результатом.

Метод вернет ошибку с описанием и кодом:
1) если желаемый номер меньше 1 или больше количества чисел в списке; 
2) если в списке чисел имелось хотя одно не целое число; 
3) если файл пустой или если список в нем не начинается с ячейки А1;
4) если у файла формат не ```xlsx```.
